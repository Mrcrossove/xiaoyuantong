# 首页入口与底部导航 UI 优化

## 任务目标

优化小程序首页六个功能入口和底部 TabBar 的视觉效果，统一图标语言、减轻按钮厚重感，并使用 rpx 保证不同手机尺寸下的适配。

## 设计参考说明

参考用户提供的当前截图，主要问题是图标圆形背景过重、视觉拥挤、底部导航选中和未选中状态不够精致。本次按轻量校园生活风处理：白底、浅阴影、线性图标、明确选中态。

## 页面模块拆解

1. 首页功能入口：六宫格入口、图标、文字、选中态。
2. 底部 TabBar：首页、创业店铺、消息、我的。
3. 公共样式：全局 tabbar 和 tab-icon。

## 技术方案

1. 不引入远程图标，避免小程序审核、网络加载、包体和版权风险。
2. 使用 CSS 绘制统一线性图标，保留当前 WXML 结构。
3. 使用 rpx 完成尺寸、间距、圆角、阴影适配。
4. 将底部 Tab 选中态改为蓝色胶囊底和半填充图标，未选中保持浅灰线性。

## 实现方案

1. 调整 `pages/index/index.wxss` 中 `.services`、`.service-item`、`.service-icon` 和六个入口图标样式。
2. 调整 `app.wxss` 中 `.tabbar`、`.tab`、`.tab-icon` 公共样式。
3. 补充开发规则：必要时可选择安装或使用合适的 skill 完成任务。
