# 首页与帖子详情页

## 任务目标

基于提供的截图，实现校园平台小程序的首页和帖子详情页静态前端原型，完成以下内容：

- 首页视觉结构复刻
- 帖子详情页视觉结构复刻
- 首页右下角发布菜单交互
- 帖子详情页联系方式卡片弹窗
- 使用可扩展 mock 数据组织页面内容

## 设计参考说明

- 首页参考用户提供的首页截图
- 帖子详情页参考用户提供的帖子详情截图
- 首页功能入口固定展示 6 个
- 点击首页右下角加号展示发布类型菜单
- 点击帖子详情页“点击获取联系方式”展示联系卡片弹窗
- 当前阶段先做静态复刻，不接真实接口

## 页面模块拆解

### 首页

- 顶部导航栏
- 校区与搜索区
- Banner 轮播区
- 六宫格服务入口区
- 统一样式的帖子列表
- 右下角悬浮发布按钮
- 发布类型浮层菜单
- 底部 TabBar

### 帖子详情页

- 顶部导航栏
- 发帖人信息区
- 帖子标题与正文区
- 图片宫格区
- 联系方式获取入口
- 点赞/评论/收藏/转发操作栏
- 评论区空状态
- 底部输入框
- 联系方式卡片弹窗

## 技术方案

- 使用原生 HTML + CSS + JavaScript 搭建静态原型
- 使用单页面双视图方案，在首页与详情页之间切换
- 使用统一 mock 数据驱动首页帖子卡片和详情页内容
- 使用 CSS 自定义属性管理主题色、阴影、圆角和间距
- 使用纯前端状态控制发布菜单和联系方式弹窗显示

## 实现方案

1. 创建静态原型入口文件
2. 先完成整体布局骨架和移动端容器
3. 实现首页模块与帖子卡片列表
4. 实现帖子详情页及页面切换
5. 实现发布菜单展开交互
6. 实现联系方式卡片弹窗
7. 补齐 mock 数据、空状态、按钮状态和样式细节
8. 进行结构自检和运行验收
